# Bookmark Breadcrumb

`BookmarkBreadcrumb` anchors to the content pane and renders the ribbon-style navigation that sits at the top of the white card.

```tsx
import BookmarkBreadcrumb from "@/components/breadcrumb/BookmarkBreadcrumb";

<BookmarkBreadcrumb
  items={[
    { label: "home", href: "/" },
    { label: "mark2", href: "/mark2" },
    { label: "Shaolin Scrolls" },
  ]}
/>;
```
