# Breadcrumb

`Breadcrumb` renders a parchment-style nav that sticks just below the site header.

```tsx
import Breadcrumb from "@/components/ui/breadcrumb";

<Breadcrumb
  items={[
    { label: "home", href: "/" },
    { label: "mark2", href: "/mark2" },
    { label: "Shaolin Scrolls" },
  ]}
  sticky
/>;
```

If a layout needs extra clearance (for example, to float over cards), tune the sticky offset with `nudgeY`.

```tsx
import Breadcrumb from "@/components/ui/breadcrumb";

<Breadcrumb
  items={[
    { label: "home", href: "/" },
    { label: "mark2", href: "/mark2" },
    { label: "Shaolin Scrolls" },
  ]}
  sticky
  nudgeY="-12px"
/>;
```
